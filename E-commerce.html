API E-Commerce

Para el desarrollo de esta API nos pedían que un usuario nuevo pudiera crear una cuenta y editarla, si se requería, que permitiera el inicio de sesión con encriptación de contraseñas para su seguridad, así como poder deshabilitar la cuenta por parte de dicho usuario. También nos pedían crear productos, actualizarlos, deshabilitarlos, buscarlos por categoría o por ID, se pedía la creación de un carrito de compras donde el usuario pudiera ver los productos que desea comprar, etc.

Para esta aplicación se usó:

  • Node.js
  • Expess
  • Sequelize
  • PostgreSQL
  • JWT
