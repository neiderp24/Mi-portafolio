Pokedex

Para el desarrollo de esta aplicación web se pedía que el usuario pudiera iniciar sesión con su nombre y le diera un mensaje de bienvenida, también se pedía buscar los pokemones por su ID, por su nombre y por su tipo.

Para esta aplicación se usó:

  • React
  • Router-dom
  • Redux
  • CSS
  • HTML
  • Consumo de API
