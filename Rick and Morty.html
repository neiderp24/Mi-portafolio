App Rick and Morty

Para el desarrollo de esta aplicación web se pedía que el usuario pudiera buscar los personajes de la serie por su ID, dando detalles precisos de dicho personaje. También se pedía buscar por su dimensión a la que pertenecen y además información como la cantidad de residentes de ese universo, etc.

Para esta aplicación se usó:

  • React
  • Axios
  • CSS
  • Consumo de API
